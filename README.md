This project, Process Scheduler Simulator with Cybersecurity, is developed as both a teaching aid and a research platform. It merges traditional operating system scheduling approaches—such as Round Robin and Priority Scheduling—with a dynamic security framework capable of observing and responding to potentially harmful process behavior, including simulated Denial of Service (DoS) cases.

Our system equips users with a live dashboard, providing clear visualizations (Gantt charts), real-time alerts, and access to key performance statistics (such as average wait, turnaround, and detection rates). The platform's security analysis leverages both threshold-based monitoring and a machine learning model (Isolation Forest) for anomaly detection, offering a layered threat detection capability.

Built using Python (Flask backend with scikit-learn for ML) and a modern HTML/JS frontend, the simulator also incorporates REST APIs and WebSocket-based live communications for a seamless user experience. Docker support ensures a smooth deployment regardless of environment.

Intended for learners, researchers, and creators, this project encourages interactive exploration of how scheduling decisions and security systems intertwine in modern computing, making it a practical foundation for further academic or technical innovation.
